Submodules

基本使用

  • 添加一个子模块：
    • git submodule add https://github.com/username/repos.git path/to/dir
  • 加载子模块：
    • git submodule init
    • git submodule update
  • 更新子模块：
    • 可以到子模块执行更新
    • 或者：git submodule update –remote
